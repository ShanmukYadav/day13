## 📝 Part C — `interview_answers.py` (20%)

### Q1 — `.loc[]` vs `.iloc[]`

| | `.loc[]` | `.iloc[]` |
|---|---|---|
| Basis | **Label** (index values) | **Integer position** (0-based) |
| Slice end | **Inclusive** | **Exclusive** |
| Works with | Any index type | Always 0, 1, 2 … |

**Example with integer index `[0,1,2,3,4]`:**
- `df.loc[0:3]` → rows 0, 1, 2, **3** (inclusive)
- `df.iloc[0:3]` → rows 0, 1, **2** (exclusive)

**Example with string index `['a','b','c','d','e']`:**
- `df.loc['a':'d']` → `a`, `b`, `c`, **d** (inclusive)
- `df.iloc[0:3]` → positions 0, 1, 2 → `a`, `b`, **c**

### Q2 — `analyze_csv(filepath)` function

Returns a dict with `num_rows`, `num_cols`, `numeric_cols`, `categorical_cols`, `null_counts`, `memory_mb`. Also prints the full First 5 Minutes checklist.

### Q3 — 3 Bugs Fixed

| Bug | Original | Fix |
|---|---|---|
| **Bug 1** | `and` with Series → `ValueError` | Use `&` with parentheses |
| **Bug 2** | Chained indexing `df["col"][0] = val` | Use `.loc[0, "col"] = val` |
| **Bug 3** | `iloc[0:2]` expecting 3 rows | Use `iloc[0:3]` (end is exclusive) |

## 🧠 Key Concepts Covered

- **Pandas Series vs DataFrame** — creation from dicts, list of dicts, NumPy arrays
- **`.loc[]`** — label-based, inclusive slicing, boolean mask filtering, safe assignment
- **`.iloc[]`** — position-based, exclusive slicing, row/column range selection
- **Boolean conditions** — `&`, `|` operators with parentheses (never Python `and`/`or`)
- **`df.query()`** — SQL-like string filtering for readable conditions
- **`df.nlargest()` / `df.nsmallest()`** — fast outlier/top-N detection
- **`pd.concat()`** — combining multiple DataFrames row-wise
- **CSV I/O** — `df.to_csv()` with `index=False`, `pd.read_csv()`
- **Data quality** — missing values, duplicates, memory, unique value analysis
